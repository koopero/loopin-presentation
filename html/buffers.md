# Buffers

``` yaml
# Send the buffer to display
show: pond
```

``` yaml
# Send the buffer to display
show: tentacles
```


``` yaml
# Change render resolution.
buffer:
  tentacles:
    width: 160
    height: 90

show: pondcreature
```

``` yaml
# Back to HD
buffer:
  tentacles: { width: 1920, height: 1080 }
  pondcreature: { width: 1920, height: 1080 }


show: pondcreature
```
